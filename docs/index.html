Simplify movie night.

EasyMovie helps you pick movies from your Kodi library. Answer a few quick questions — genre, rating, runtime — and get a curated random selection. No more scrolling through hundreds of titles.

Download Latest Release View on GitHub

Key Features

Filter Wizard

Quick questions for genre, rating, runtime, year, and score narrow your library to movies that match your mood

Browse Mode

See your curated selection in a visual layout. Five view styles to choose from.

Playlist Mode

One click builds a movie marathon. Sit back and let EasyMovie decide.

Movie Set Awareness

Automatically suggests the next unwatched movie in a collection and prompts to continue the set

4 Color Themes

Golden Hour, Ultraviolet, Ember, and Nightfall accent colors for every view and dialog

Re-roll & Surprise Me

Don't like the selection? Get a fresh set. Or let EasyMovie pick one at random.

Smart Re-suggestion

Avoids suggesting the same movies too frequently with a configurable cooldown window

Clone Support

Multiple EasyMovie instances with different filter presets on your home screen
